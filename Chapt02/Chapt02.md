# ２章　プログラムの書き方　補足
## 2.2　書き方の慣習
### 2.2.1　関数の書き方
#### K&Rスタイル
```C
int main(void) {
    return 0;
}
```

- 行数が少なく、コンパクト
- 横に詰まるのでスクロールが少ない
- C言語の伝統的スタイル（標準ライブラリなどもこれが多い）
#### Allmanスタイル
```C
int main(void)
{
    return 0;
}
```
- 波括弧が縦に揃うので視認性が良い
- ブロック構造が分かりやすい
- 初学者には読みやすいことが多い
